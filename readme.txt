== Description ==
hm-woocommerce-coming-soon lets you to enable coming soon mode for products with time to launch it. Products will be automatically launched on that time, and will show a countdown timer until that time overs.

It removes the “add to cart” button from product when product is in coming soon mode.

You can also customize Clock colors, and it has ability to hide/show price and tabs when product is in coming soon mode.





== Installation ==

1. Add plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to widget secction and add it any sidebar
1. Fill the state and city (Required)
1. check other option if you want to show other information
